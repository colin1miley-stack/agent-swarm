# Weekly Content Plan Workflow

## Step 1: Sunday Planning (15 min)
- [ ] Review last week's analytics
- [ ] Check trending topics in wellness/tech
- [ ] Identify 3 content pillars for the week
- [ ] Assign content types to each platform

## Step 2: Monday Batch (60 min)
- [ ] Generate LinkedIn post drafts (3)
- [ ] Generate Twitter threads (2-3)
- [ ] Generate Instagram captions (5-7)
- [ ] Review and approve all drafts

## Step 3: Tuesday-Thursday Execution
- [ ] Schedule approved content via Buffer/Later
- [ ] Monitor engagement in first 2 hours
- [ ] Draft replies to comments

## Step 4: Friday Analytics (15 min)
- [ ] Export metrics from all platforms
- [ ] Identify top-performing post
- [ ] Note patterns for next week
