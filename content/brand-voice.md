# Brand Voice Guidelines

## Core Identity
- **What you are:** A wellness tech founder building in public
- **What you believe:** Wellness should be personalized, transparent, and accessible
- **Your edge:** 15+ years enterprise SaaS + AI-powered systems

## Tone Dimensions
| Dimension | Setting |
|-----------|---------|
| Formality | Semi-professional (70/30) |
| Humor | Dry wit, occasional |
| Vulnerability | High — share failures openly |
| Confidence | High — but earned, not arrogant |
| Technical depth | Medium — explain concepts clearly |

## Voice Checklist
Before publishing, check:
- [ ] Does this sound like a real person?
- [ ] Is there a clear point of view?
- [ ] Would I say this to a friend?
- [ ] Is there specific detail (not generic advice)?
- [ ] Does it invite conversation?

## Banned Words
- Leverage
- Navigate
- Unlock
- Empower
- Delve
- Synergy
- Optimize (unless technical context)
- Scale (unless genuine growth context)
- Disrupt
- Innovative

## Filler Replacements
| Instead of... | Say... |
|---------------|--------|
| "In today's world..." | Start with the point |
| "It's important to..." | Just say it |
| "At the end of the day..." | Cut it |
| "I believe that..." | State it as fact |
| "The reality is..." | Just state the reality |
